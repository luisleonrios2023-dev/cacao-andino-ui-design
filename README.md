# 🍫 Cacao Andino

Sitio web responsive para una pequeña pastelería artesanal que combina postres tradicionales de Ecuador y España.

El proyecto presenta el catálogo de productos, la historia de la pastelería y sus datos de contacto mediante una experiencia visual cálida y adaptada a diferentes dispositivos.

## ✨ Funcionalidades

* Diseño responsive para móviles, tabletas y escritorio.
* Catálogo organizado por categorías.
* Filtrado dinámico de productos mediante JavaScript.
* Cabecera y pie de página reutilizables cargados con `fetch`.
* Navegación multipágina.
* Formulario de contacto.
* Publicación mediante GitHub Pages.

## 🛠️ Tecnologías

* HTML5
* CSS3
* JavaScript
* Git
* GitHub Pages

## 📂 Secciones

* **Inicio:** presentación de la pastelería y productos destacados.
* **Productos:** catálogo organizado por categorías.
* **Pastelería:** historia, valores y propuesta del negocio.
* **Contacto:** información de contacto y formulario.

## 💡 Objetivo del proyecto

El objetivo fue desarrollar una página web completa para un pequeño negocio local, aplicando diseño responsive, organización modular de estilos y reutilización de componentes mediante JavaScript.
